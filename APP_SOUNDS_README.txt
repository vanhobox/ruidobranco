# Sound Files Required

Place the following audio files in `app/src/main/res/raw/`:

- uterine.ogg (or .mp3)
- rain.ogg (or .mp3)
- ocean.ogg (or .mp3)
- waterfall.ogg (or .mp3)
- fan.ogg (or .mp3)
- humidifier.ogg (or .mp3)
- radio_static.ogg (or .mp3)
- white_noise.ogg (or .mp3)

These should be royalty-free looping audio tracks in OGG or MP3 format.
Recommended: 30-60 second seamless loops at 128-256kbps bitrate.
